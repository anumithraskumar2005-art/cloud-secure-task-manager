Cloud-Based Secure Task Management Dashboard

This project demonstrates a secure web application deployed using Cloud, DevOps, and Cybersecurity best practices.

Custom VPC, public/private subnets, route tables, ALB, and domain setup.

DevOps & Automation

GitHub, Docker, CI/CD pipeline, automated deployment, and backup scripts.

Monitoring & Logging

Application logs, failed login tracking, health checks, and CloudWatch alerts.
